
# Keeper (Keeping notes)

A simple note storing book where a user can perform operations:-   
1- Add a note with a title which will be displayed on the screen.  
2- Delete a note.   
3- Search for a note based on the content.  

**Tech stack:**  
1- HTML   
2- CSS  
3- JavaScript
